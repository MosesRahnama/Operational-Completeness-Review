# Logs

Purpose: Canonical CSV logs that define the review metadata and file inventory.
Contents: File list and short descriptions (see below).
- `review_log.csv` — master per‑file log with category, failure mode, and paths.
- `manifest_all_files.csv` — raw inventory of all files discovered during the review.
- `repo_manifest.csv` — complete manifest of every file in this repo with purpose and metadata.
